# 2023-11 QQ 空间相册导出工具，变现 2W

## 背景

QQ 空间存放了大量历史照片，希望能批量导出。

## 可行性

QQ 空间用户量巨大，故需求也大，而且有导出需求的基本是成年人，付费能力也高。参考另一个火爆项目“老照片修复”，笔者认为是同一级别的需求。
