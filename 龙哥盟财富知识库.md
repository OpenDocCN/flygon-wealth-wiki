## 下载

### Docker

```
docker pull apachecn0/flygon-wealth-wiki
docker run -tid -p <port>:80 apachecn0/flygon-wealth-wiki
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-wealth-wiki
flygon-wealth-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
